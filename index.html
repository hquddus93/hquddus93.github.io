TrailMix

Live Site

|

Github

A clone of Netflix, TrailMix is a fullstack application built using Ruby on Rails for the backend, React/Redux for the frontend, PostreSQL for the database, and AWS S3 for video/image storage. Trailix has complete frontend/backend user authentication which allows users to make an account, sign into an existing account, or try the demo account for hassle free viewing. TrailMix has all videos sorted by genre, and users can add/delete video from their personal list. Users can hover over videos to watch them play or click on them to view them in fullscreen mode. TrailMix also boasts a fully functioning search bar and not to mention top-notch video content.

OwnProps.match

Live Site

|

Github

Are you new to programming and are looking for a pair programmer to guide you through some beginner projects? Or maybe you're a seasoned coder looking for a qualified partner to help you take your project to the next level. Look no further, Ownprops.match is here for you! Ownprops.match is a fullstack application built using the MERN stack. Myself and three other engineers came together to create this excellent service for the coding community. When users create an account they fill out a form, providing information on what their primary coding language is, what their goal is, what their experience is, as well as other user authentitcation information. Once signed up, users can swipe through other users' profiles, displayed as cards, and either select or reject people they are intersted in working with. If a user matches with another user, both parties will receive a notification and will be given their match's email in order to initiate contact.

CSSMR is a fun, interactive site built using JavaScript, HTML5, and CSS3. CSSMR is a play on ASMR and provides users with a multitude of satisfying sounds, as well as trivia knowledge! Users can hover over containers all over the page to hear the sounds, or they can press certain keyboard keys to create the sounds on their own.
